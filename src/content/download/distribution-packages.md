---
title: "Distribution Packages"
---

### Fedora

Install from the `william341/hummingbird` COPR repository.

```bash
sudo dnf copr enable william341/hummingbird
sudo dnf install hummingbird
```

### Arch Linux

Install the `hummingbird-player` package from the AUR.

```bash
paru -S hummingbird-player
```

### NixOS

Use the flake provided in the main repository. Hummingbird is not yet available in Nixpkgs.
